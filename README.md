AI SEO Strategy & Content Engine

An automated workflow for high-scale organic growth.

Workflow

Ingest: Import .csv keyword exports from traditional SEO tools.

Cluster: Use semantic embeddings to group keywords by user intent.

Generate: Trigger LLM-based brief generation to create SEO-optimized content outlines.

Scale: Output structured markdown briefs for content teams or automated drafts.

Setup

Add OPENAI_API_KEY to your environment variables.

Place keyword data in data/keywords.csv.

Run python main.py.
